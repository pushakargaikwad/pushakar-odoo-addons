Remove Time from Quotation Print

Removes Quotation Creation Time from Quotation and Sales Order Print

By default Odoo displays displays both Date and Time of creation on the Quotation and Sales Order print

In some companies, the time of creation of quotation or Sales Order is not expected to be included on the Print. Displaying just the Date is enough

This module removes time from "Date and Time" which is displayed on Quotation and Sales Order Print

Before

Both Date and time fields are visible

After

Only Date in "Date Ordered" is visible

Updated on both Quotation and Sales Order prints
