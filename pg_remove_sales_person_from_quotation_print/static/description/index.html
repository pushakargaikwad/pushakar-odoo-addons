Remove Salesperson from Quotation Print

Removes from both Quotation and Sales Order

By default Odoo displays displays Salesperson name on the Quotation and Sales Order

In some companies, the quotation is generated in the name of company and it is not expected to include the name of person who created the Quotation.

This module Changes removes the Salesperson field from the print of Quotation and Sales Order.

Before

Salesperson field is visible

After

Salesperson field is removed

Updated on both Quotation and Sales Order prints
