Update Cost/Unit Price when Product changes on Purchase Order

Onchange product updates the unit price on Purchase Order Line

By default Odoo gets the cost price for product only if vendor price is set. But in somecase we want to get the cost price for product even if vendor price is not set, that is the default cost price configured on product level.

This module displays the get the cost price set on product level even if vendor price is not set on purchase order line.
